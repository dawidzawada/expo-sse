# Example App

Expo app demonstrating `@dawidzawada/expo-sse` with auth and token refresh.

## Running

1. Start the [mock server](../server/):

   ```bash
   cd apps/server && bun run dev
   ```

2. Start the example app with Expo Go:

   ```bash
   cd apps/example && bun run start
   ```

3. Press `i` or `a` to open on simulator/emulator or scan QR. runs on Expo Go.
